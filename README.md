# Knuckle



## Commands

* Run tests: `$ npm run test`
* Format code: `$ npm run format`
* Lint code: `$ npm run lint`
* Build: `$ npm run build`
* Add contributor: `$ npm run contributor add <github-username> <contribution-type>`
* Deploy: `$ npm run deploy`
